# Node Farm 🐮🌱

A bootcamp coursework project built with **Node.js** to practice fundamental backend logic.  
It features a custom HTTP server, routing, dynamic HTML templating, and slug-based navigation using the `slugify` package.

---

## 🚀 Live Demo

## [View Demo](https://node-farm-klxv.onrender.com/)

## 📂 Project Structure

- `server.js` → Main server file using `http.createServer()`
- `templates/` → HTML templates for overview, product, and card pages
- `dev-data/data.json` → Sample product data
- `modules/replaceTemplate.js` → Utility for injecting data into templates

---

## ⚙️ Installation

Clone the repository and install dependencies:

```bash
git clone https://github.com/NwankwoMichael/Node-farm.git
cd Node-farm
npm install
```
